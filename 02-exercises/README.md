## 📚 מבנה התרגילים

תרגילים אלו מחולקים ל-3 רמות קושי:

### [🟢 Basic - תרגילים בסיסיים](basic/)
למי שמתחיל עכשיו עם MongoDB:
- תרגיל 1: חיבור ויצירת collection
- תרגיל 2: הוספת מסמכים
- תרגיל 3: קריאת מסמכים
- תרגיל 4: עדכון פשוט
- תרגיל 5: מחיקת מסמכים

### [🟡 Intermediate - תרגילים בינוניים](intermediate/)
למי שיודע את היסודות:
- תרגיל 6: queries מורכבים
- תרגיל 7: aggregation בסיסי
- תרגיל 8: עבודה עם arrays
- תרגיל 9: indexes
- תרגיל 10: transactions

### [🔴 Advanced - תרגילים מתקדמים](advanced/)
למי שרוצה אתגר:
- תרגיל 11: aggregation pipeline מורכב
- תרגיל 12: ייצוא/ייבוא נתונים
- תרגיל 13: performance optimization
- תרגיל 14: data modeling
- תרגיל 15: פרויקט מיני מלא

## 🎯 איך לעבוד עם התרגילים?

### 1. התקנה ראשונית
```bash
cd 02-exercises
npm install
```

### 2. בחר רמת קושי
```bash
# בסיסי
cd basic

# בינוני
cd intermediate

# מתקדם
cd advanced
```

### 3. פתור תרגיל
כל תרגיל כולל:
- 📖 README עם הוראות
- 🎯 קובץ התחלה עם TODO
- ✅ פתרון מלא (בתיקייה solutions/)

### 4. בדוק את עצמך
```bash
npm run test
```

## 💡 טיפים

1. **נסה בעצמך קודם** - אל תקפוץ ישר לפתרון
